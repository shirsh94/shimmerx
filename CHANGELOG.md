## 0.0.1

### Initial Release

**Features:**
- 🎨 **10 Animation Styles** - wave, pulse, glow, fade, slide, diagonal, circular, radial, breathing, aurora
- 🌈 **Gradient Support** - Full support for LinearGradient, RadialGradient, SweepGradient, and custom gradients
- 🎭 **Global Theming** - ShimmerTheme and ShimmerThemeData for global configuration
- 🧩 **Platform Presets** - 8 ready-to-use presets (YouTube, Instagram, Amazon, Netflix, Spotify, Chat, Settings, Dashboard)
- ⭐ **Animation Control** - ShimmerController with pause, resume, stop, restart, and reverse methods
- 🔧 **Extension Methods** - shimmer(), shimmerThemed(), shimmerWith() for quick implementation
- 📱 **Material 3 Ready** - Modern UI components with Material 3 design support

**Documentation:**
- Comprehensive README.md with usage examples
- MIT License added
